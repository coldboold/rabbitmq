## rabbitmq
> 为什么要学习rabbitmq

- 能够与SpringAMQP完美整合,api丰富
- 集群模式丰富,表达式匹配,HA模式,镜像队列模式
- 保证数据不丢失的前提下可做到高可靠性，可用性
